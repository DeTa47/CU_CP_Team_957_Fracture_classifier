How to run the files:

For frontend:
	- open the folder 'fe' in VS Code or any other code editor.
	- open the terminal and run 'npm run dev'
	- if there is an error for dependecies, run 'npm install' and then run 'npm run dev'
	
For backend:
	- open folder 'be' in your desire code editor
	- open terminal and run 'python main.py'
	- if there is error due to libraries not available, install them using commnd 'pip install <name>'
